# webmis-python
采用Python + Redis + MariaDB开发的轻量级HMVC基础框架，目录结构清晰，支持CLI方式访问资料方便执行定时脚本。包括HMVC模块化管理、自动路由、CLI命令行、Socket通信、redis缓存、Token机制等功能，提供支付宝、微信、文件上传、图像处理、二维码等常用类。

## 安装
```bash
$ git clone https://github.com/webmiss/webmis-flask.git
$ cd webmis-flask
$ ./bash install
```

## 运行
```bash
# Linux、MacOS
./bash serve
# Windows
.\cmd serve
```
